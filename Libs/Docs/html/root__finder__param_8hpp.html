Find the roots of a function in a given interval.

This is an overloaded member function, provided for convenience. It differs from the above function only in what argument(s) it accepts.

Find the roots of a function f(x, k) in a given interval [xa, xb] using the specified method. k is a parameter of the function and is simply passed to the function. Works by first bracketing the roots and then applying the method on every sub-interval.

Parameters
[in]fPointer to the function.
[in]dfdxPointer to the derivative of the function.
[in]paramAdditional function parameter.
[in]xaLower bound of the interval.
[in]xbUpper bound of the interval.
[in]tolx-tolerance.
[out]rootsArray with the roots of f(x, k).
[out]nRootsThe number of roots found.
[in]NThe number of sub-intervals.
[in]methodThe root finding method. Accepted values are: bisection, falsePosition, secant, newton.
Returns
flag
Return values
0Success.
1Too many steps.
2Initial interval doesn't contain any root.
Exceptions
std::invalid_argumentThrown if N > 128.
std::invalid_argumentThrown if method is not among the accepted values.
std::runtime_errorThrown if roots can't be found inside the interval.
std::runtime_errorThrown if one of the root finders exceeded the maximum number of steps.

Bracket the roots of a function in a given interval [xa, xb].

Works by subdividing the interval in a number of sub-intervals and checking if the function changes sign (an odd number of times) over this interval. If it does, then the interval contains (at least) one root.

Parameters
[in]fPointer to the function.
[in]paramAdditional function parameter.
[in]xaLower bound of the interval.
[in]xbUpper bound of the interval.
[out]xLArray with the lower bound of the sub-interval containing a root.
[out]xRArray with the upper bound of the sub-interval containing a root.
[in]NThe number of sub-intervals.
[out]nRootsThe number of roots found.

Find the root of a function f(x, k) in a given interval [xa, xb] using Newton-Raphson method. k is simply passed to the function.

This is an overloaded member function, provided for convenience. It differs from the above function only in what argument(s) it accepts.

Parameters
[in]fPointer to the function.
[in]dfdxPointer to the derivative of the function.
